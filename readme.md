eGroups

⚠️ Half-assed prototype. Merul: let me know if I can assis you further.

A lightweight app for creating and managing invite-based groups.
Features

Create a group — set a name, bio, and avatar; choose open or admin-approved membership
Invite people — share a QR code or link to let others request to join
Manage members — approve or remove members, promote others to admin
Member view — see all groups you belong to and their members
Group dashboard — manage your own groups or browse ones you've joined

How it works

Log in to eGroups
Create a group and configure its settings
Share the invite link or QR code
Approve (or auto-accept) incoming join requests
Manage your group and its members from your dashboard
